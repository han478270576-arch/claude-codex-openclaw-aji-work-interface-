Ajiclaw • OpenClaw Control Plane

Aji Super Individual Studio

这是阿吉的 OpenClaw 控制平面公开入口。根域名只提供说明，不直接暴露私有控制台 token。 真正的生产与测试访问通过独立子域名进入，并要求已配对设备身份。

Production

prod.ajiclaw.com

生产环境入口。用于正式运行中的 OpenClaw 控制台与网关管理。

Testing

test.ajiclaw.com

测试环境入口。用于验证改动、升级和新配置，不直接影响生产。

Operations

Terminal First

终端统一入口命令为 openclaw-codex-claude，适合日常运维与排障。

$ openclaw-codex-claude
$ bash ./bin/openclawctl.sh status
$ bash ./bin/openclawctl.sh start prod
$ bash ./bin/openclawctl.sh start test

提示：如果你是首次使用新设备，请先完成设备配对；若你没有带 token 的私有入口书签，请通过已授权终端获取。
